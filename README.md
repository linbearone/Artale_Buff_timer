# Artale_Buff_timer
簡介：
一個自用的技能時間計時器
製作過程使用chatgpt協作

－－－－－－－－－－－－－－－－－－－－
功能敘述：
核心功能為選擇鍵盤上的一個按鍵
當它被按下時觸發計時器的倒數計時
提供懸浮窗顯示技能圖示及倒數時間
可以設定在技能結束前發出音效提醒

計時器無法主動對應被消技導致的技能結束
或是其他隊友分享的增益效果
折衷方案可以將觸發按鍵綁定在空按鍵上
在獲得增益效果時再手動開始計時

－－－－－－－－－－－－－－－－－－－－
備註事項：
計時器可能的浮動延遲為500ms
連續快速按下觸發按鍵也會導致整體計時滯後
提示音中的beep音效與系統音效音量有關
沒有聲音時可以嘗試使用midi音效

如果小工具的按鍵沒有反應
可以在exe檔案點擊右鍵>內容>相容性 中
勾選以系統管理員執行此程式的選項
自用環境為windows11
不確定其他作業版本是否正常運行
